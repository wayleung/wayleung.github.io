题目：

请实现一个函数，把字符串 s 中的每个空格替换成”%20”。

示例 1：

输入：s = “We are happy.”
输出：”We%20are%20happy.”  

限制：

0 <= s 的长度 <= 10000

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof

然后再上一下我的提交结果：

我的解题思路：

1.首先遍历记录下空格字符的数目，原字符数组大小+空格字符的数目*（替换后的字符串-1）得出新数组的大小

2.遍历字符串的原字符数组遇到空格字符的话  新数组i位置以及后面的元素为 %20还有 修改i的位置 直至遍历完成

3.最后返回新数组组成的字符串
